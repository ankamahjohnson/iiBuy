# E-Makola 
A fully functional e-commerce web app with Paystack API integration built to sell gadgets to millennials and Gen-Z consumers across Africa, featuring a complete shopping cart system, live payment method (Momo & Banking visa card), dark/light mode, and persistent data using localStorage


## Live Demo
View Live 🌐
https://ankamahjohnson.github.io/E-Makola/


## Features
### 🛍️ Product Discovery
- 18 quality products across 6 categories
- Real-time search bar that filters products instantly
- Category filter buttons
- Star ratings displayed on every product card
- Product detail modal with full description and rating

### 🛒 Shopping Cart System
- Add products to cart with a single click
- Remove products directly from the landing page
- Real-time cart button counter
- Persistent cart saved in localStorage (survives page refresh)
- Fully featured cart modal:
  - Increase or decrease quantity per item
  - Remove items from cart
  - Dynamic price updates based on quantity
  - Automatic subtotal calculation

### 💰 Checkout & Payment
- User details form with robust validation (name, email, phone)
- Shipping calculator with regional rates
- Dynamic subtotal, shipping fee, and grand total calculation
- Full **Paystack** payment integration (Test Mode)
- Checkout button validates form before opening payment modal

### 👤 User Profile & Orders
- Customer profile page with localStorage persistence
- Profile data automatically fills the checkout form
- Order history saved after every successful purchase
- Order summary modal after payment (shows customer name, items & quantities)
- "OK" button clears cart and resets the experience

### 🎨 User Experience & Polish
- Beautiful landing page with four structured sections
- Toast notifications for every action (add to cart, remove, success, etc.)
- Dark and Light mode toggle with localStorage persistence
- All modals close when clicking on the overlay
- Fully responsive design (mobile-first)
  
### 📱 Responsiveness
- Optimized mobile experience with vertical stacking
- Clean grid layout on tablets and desktops
- Fully functional across all screen sizes
